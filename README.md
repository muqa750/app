# Sizeme Flutter App

Luxury starter app for a fashion store with:
- Clean architecture folders: `Domain`, `Data`, `Presentation`
- Arabic first, English second
- Luxury home screen (white/black/gold theme)
- Bottom navigation with Home, Favorites, Cart, Categories, Menu
- Side drawer (hamburger menu)
- Light/Dark mode toggle

## Important
Flutter SDK is not currently available in this environment, so native folders were not auto-generated yet.

To generate full Android and iOS project files on your machine:

1. Install Flutter and run:
   - `flutter doctor`
2. In this project folder run:
   - `flutter create .`
3. Restore dependencies:
   - `flutter pub get`
4. Run app:
   - `flutter run`

> `flutter create .` keeps your `lib` files and generates full `android` and `ios` folders correctly.
